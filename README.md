## Deployment

Download PIP get-pip.py
```bash
  curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py
```
Installing PIP on Windows
```bash
  python get-pip.py
```
Check Python version
```bash
  py --version
```
Ensure you can run pip from the command line
```bash
  py -m pip --version
```
